2、函数

就是一个工具， 为了完成某个功能的一段代码

3、定义一个函数

方式一、   
function   函数名(){  //   可以在函数的上面或下面进行函数的调用  --- 函数的提升（将函数自动提升到程序的最顶端） 
     函数体;
}

方式二、 表达式定义法
var fn = function(){  //   匿名函数   ---- fn代表了这个函数的名称    必须在先定义 后调用（不会实现函数提升）

}

5、函数的参数

定义一个函数时， 传递参数   （让程序更灵活）
function 函数名( 参数列表 ){  参数可以有一个或多个

}
参数分类：
形式参数（形参）：
     形参出现在函数定义中
     形参必须是一个变量
     如果形参的个数大于实参的个数，多余的形参值为undefined
实际参数（实参）
     实参出现在函数调用中
     实参可以是变量、常量、表达式，但必须有值
     将实参的值传递给形参  
     实参的值传递给形参时，实参的顺序和形参变量的顺序一致 （一一对应）
     如果实参的个数大于形参的个数，多余的实参自动舍去

6、函数的返回值

当一个函数的结果需要用作其它用处时，函数需要返回一个值，这个值就叫做函数的返回值
函数的返回值通过return来返回
一个可以没有返回值
一个函数如果有返回值，只能有一个
一个函数如果遇到了return，return后面的代码不再执行 

7、变量的作用域

全局变量和局部变量
全局变量： 变量在整个程序内都有效
     在函数外面定义的变量是全局的
     在函数内部没有通过var定义的变量 是全局变量（隐式全局变量）
局部变量 ：变量只在某个函数体内部有效，函数调用后，局部变量自动销毁
     所有的形参都是局部变量
     局部变量的优先权高于全局变量


8、变量提升

在函数体内部用var声明的变量，会将变量提升到函数的最顶端，只提升声明  但不赋值

function fn(){
     alert(m);
     var m = 10;
} 等价
function fn(){
     var m;
     alert(m);
     m = 10;
}
