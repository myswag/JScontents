2、什么是xss?


跨站脚本攻击  cross  site  script
为了不和层叠样式表重名，顾将css跨站脚本攻击 改为 xss
什么是脚本攻击：
     专业的程序员会在  浏览器的文本框中输入一些非法信息 ：比如  超链接 等，  服务器会识别这些非法信息，这就是脚本攻击。

第一种解决办法：前端正则做验证 （不能包含特殊字符）
前端验证有弊端：  内行人会将页面的表单信息拷贝到其它页面上，也可以实现脚本攻击

第二中解决办法：服务器端做验证 （去掉不必要的标签信息）



1、什么是对象？

对象是一个整体，对外提供一些功能。
一切具有属性和方法的事物
一切具有本质特征和行为的物质
对象的两个特性：  属性和方法   window  

笔记本：
     属性  ---  大小   颜色   价格   品牌  厂家

基本类型：没有属性和方法
var num = 23;
num.index = 90;
alert( num.index  );  undefined

引用类型: 都有属性和方法  
var arr = [2,3,4,5,6];

}

2、什么是类？


类是具有相同属性和行为的一类事物的总称。
类和对象关系：
     类是对象的抽象，对象是类的实例 （类是对象的抽象化，对象是类的具象化）

3、什么是面向对象？ oop

面向过程： C语言     
面向对象： 面向对象实际上就是一种编程思想         php   java  c++  c#

高速公路长 10000米，行走速度  15米/min   走完这条路需要多长时间？

面向过程：   10000/15

面向对象：
     公路  长度属性  len =  100000

4、对象的创建（单个对象创建）

1、  var obj = new Object();  构造函数方式
2、  var obj = {  } 类似json对象的创建    （严格的json对象没有function）

5、工厂模式（批量对象创建）

优点：解决多个同类对象创建时代码重复的问题。
缺点： 工厂模式实际上就是普通函数的调用，并不是通过new关键字创建出来的，不符合对象的创建规范  （一般 一个对象都是通过new 关键字创建出来的）
      不能确定某个对象属于哪一个类

6、构造函数

为了和普通函数进行区分，构造函数的命名方式是 大驼峰 QianFengJiaoYu
构造函数中的属性叫做实例属性
构造函数中的方法叫做实例方法

优点：
     规范对象创建方式 通过new关键字
     能够确定某个对象属于哪一个类 

缺点： 同类对象的方法会被重建 （空间不共享）     a1.running ==  a2.running    false

7、原型构造函数  （所有的构造函数都有一个原型属性）

写在构造函数外面的属性叫做原型属性
写在构造函数外面的方法叫做原型方法

优点 ： 同类对象的方法被重建的问题得到解决  
缺点 ： 同类对象的属性名完全相同。

8、混合构造函数 

把属性写成实例属性
把方法写成原型方法
