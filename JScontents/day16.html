2、for..in   /  for..of
var arr = [1,2,3,4,5];
for( var i in arr ){    遍历键
     console.log(i); //下标  键
}
for(var i of arr){      遍历值
     console.log(i);
}

3、箭头函数  =>
var fun = function(){ ... }

箭头函数：  () => { 函数体 }
           (参数) => { 函数体 }   如果参数只有一个  小括号可以省略

例如： var fun = (a) => { return a+2; }
      函数调用： fun(20);

5、Array.from  将一个伪数组转成数组

     var arr = Array.from( list );
     arr.forEach( (item)=>{
          //console.log( item.innerHTML );
          item.style.backgroundColor = "pink";

7、Set和Map
Set集合方法 ： 
     add  向集合中添加值   s.add( 值 )
     delete 删除集合中某个值 返回一个布尔值 删除成功 返回true 否则返回false
     has   判断集合值是否含有某个值  返回布尔类型  
     clear  清除集合数据    没有返回值
    
     set集合特点：自动去重    不会对数据进行类型转换    ===
     数组去重 ： 把数组转成集合   再将集合转成数组   Array.from(  new Set(arr)   )
                               史上最短数组去重 ： [...new Set(arr)]

     可以直接使用forEach遍历set集合

Map 集合 ： 数据以键值对的格式存储 
     方法：
          set() :为Map集合赋值   map.set( "键","值" )
          delete() :删除 
          has() : 判断是否含有某个键  
          clear() : 清除集合数据
  可以使用for  of 来遍历map集合的值
