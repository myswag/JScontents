6、数组的赋值 (通过下标    0 表示数组的第一个数    最大下标：  数组的长度-1)

1、先定义  后赋值
var  arr = [];
arr[0] = 12；
arr[3] = 34;
arr[100] = 90;
当数组中某个下标没有赋值时，默认是undefined  

var  arr = new Array(5);
arr[0] = 90;     arr.length = 5

arr[7] = 89;     arr.length = 8;

2、定义数组时直接赋值
var arr = [3,4,5,6,778];

3、通过循环方式赋值（数组中的数据有规律）
将1--100中的偶数存入到数组中

var j = 0;
for( var i = 1 ; i <= 100 ; i++ ){
     if( i%2==0 ){
          arr[j] = i;
          j++;
     }
}

11、数组的操作方法

push()   向尾部添加（可以添加多个）
pop()    删除数组尾部的数据
shift()  删除数组首部的数据
unshift() 向数组首部添加一个或多个数据

slice(start,end)    从开始位置截取 到 结束位置处的数据，但结果不包括结束位置的数据  （不会改变原数组） 
splice(start,length)  从开始位置截取指定长度的数据  （会改变原数组）     

join()    将数组转成字符串   
concat()   数组合并 连接  
reverse()  翻转  


12、栈和堆

栈 ： 后进先出 
程序中如果定义一个变量，这个变量就会在内存中开辟一块存储单元，这个存储单元空间 就称为栈结构  
一般 栈中存放的是 基本类型的变量

堆 ： 堆中存放的数据一般都是object类型（复杂类型）    一改全改
