4、类型转换

自动转换： 
     在进行算术运算或关系运算时，纯数字的字符串会自动转成数值进行计算   
     在进行算术运算或关系运算时， true 会自动转换成1 ，false 会自动转换成 0
         null  会 自动转成 0 ， undefined 会自动转成NaN
强制转换：
     "45" + 90
     parseInt()  强制的将一个字符串类型的数据转成整数    不能正常进行转换的结果都是 NaN
     parseFloat()  强制的将一个字符串类型的数据转成小数  不能正常进行转换的结果都是NaN
     Number()    强制的将某个类型转成number类型  
     toString()  强制的将某个类型转成string类型

     toString(要转成的进制数)   将一个十进制整数 转成 2、8、16   

     扩展
     isNaN(  )  判断非数字 ：  如果括号中的数是非数字，返回true，否则返回false
     键盘输入一个数，判断奇偶 （考虑：非数字）

     eval(  )  将某个字符串转成表达式，再获取表达式的值 （去掉引号）

     eval( "alert(34)" ); --->  等价  alert(34)

6、选择结构（出口只有一个 ）

单分支选择结构（一个条件）：
if( 表达式 ){

}   或
 
if( 表达式 ){

}else{

}  
说明： 多数情况下，表达式一般是 关系表达式
执行逻辑： 同条件表达式
判断如果表达式为真，执行if语句，否则就执行else语句

说明：
     1、else可以省略
     2、如果if 或 else后面只有一条语句时，花括号可以省略的   （不建议）
     3、关于if表达式的几种情况（ 表示判断的 语句  成为布尔环境下 执行事情 ）
          非0为真，0为假   
               if(89){ alert(1) } else{ alert(2) }    // 1
          字符串为真，空字符串为假  
               if( "aaa" ){ alert(1) } else { alert(2) } // 1   ""--->假
          所有的对象为真    
          所有的函数为真    
     function fn(){

    }
    if( fn ){
         alert(1)
    }
          null和undefined 为假
