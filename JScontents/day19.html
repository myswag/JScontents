数据库的创建 ：create  database   数据库名称
 
表的创建  ：
create table  students
(
     uid  int  primary key,
     uname char(10),
     upwd char(20)
)

主键 ： 标识表中数据的唯一性  （特点：主键不能重复）  
primary key   主键  
auto_increment   自增  （一般主键设置为自增）

表中数据的操作：
添加 ： 
  insert into 表名 (字段1,字段2,....)   values(值1,值2,...)
例如：
insert into users (uname,upwd) values('小张','jack')

查询 ：
select * from 表名  where  条件  
select 字段1,字段2  from   表名  [where 条件]
例如： select * from users where  uid=1  and uname='kkk'     and  表示并且    or 或者


修改 ： 
     update  表 名   set  字段 = 新值 , 字段 = 新值   where 条件

例如：
  修改编号为2 的学生密码
   update users set upwd = "333" where uid = 2


删除  ：  delete  from  表名  [where  条件]
