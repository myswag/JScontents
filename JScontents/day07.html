5、数组操作新增的方法

indexOf()  :   
  用法：   数组.indexOf(num);   判断num在数组中是否存在，如果存在就返回num在数组中的下标，否则返回-1
function noRepeat(arr){
         var newArr = [];
         for( var i = 0 ; i < arr.length ; i++ ){
             if( newArr.indexOf( arr[i] ) == -1 ){ //判断原数组中的每一个数在新数组中是否存在
                 newArr.push( arr[i] );
             }
         }
         return newArr;
  }


forEach() : 遍历数组   ，该方法没有返回值 
用法：
     数组.forEach(function(item,index,array){
          item 表示数组中的每一项
          index  表示数组的索引
          array   表示数组本身
     })
   //实现输出数组中的素数
    arr.forEach(function(item){
         var flag = true;
         for( var i = 2 ; i < item ; i++ ){
             if( item%i ==0 ){
                 flag = false;
                 break;
             }
         }
         if( flag ){
             console.log( item );
         }
    })


filter ： 根据条件 返回一个新数组  
 用法：
     数组.filter(function(item,index,array){
               
      })

    var newArr = arr.filter(function(item){
         return item%2==0;
    })
    alert( newArr );



map  : 映射       遍历数组 ---  操作数组----返回数组
用法：
     数组.map(function(item,index){   ........  })
   //计算数组中每一个数的平方根
    var newArr = [9,16].map(function(item){
         return Math.sqrt(item);
    })
    alert( newArr );


reduce : 归并     返回结果为  最后一次操作的结果
用法：  arr.reduce(function(pre,next,item){
               pre  上一个操作返回的结果   初始值  数组中的第一个元素
                next  数组中从第二个元素开始的每一个元素  （下一个元素）
            })

     var arr = [1,2,3,4,5,6,7];
    document.write("pre" + "  " + "next<br>");
    var res = arr.reduce(function(pre,next){
         //pre  上一个操作返回的结果   初始值  数组中的第一个元素
         //next 数组中从第二个元素开始的每一个元素  （下一个元素）
         document.write( pre +"   :  "+next + "<br>");
         return pre+next;
    })
    alert( res );

6、字符串对象

定义：  
     var str = "hello";

字符串操作方法：
charAt(index)  返回某个索引位置处的字符
charCodeAt(index) 返回某个索引位置处的ASCII码值   a--97    A--65    0  ---  48
String.fromCharCode(ASCII值)   

split()   将一个字符串转成数组    join() 将数组转成字符串
str.split("字符"); 

indexOf（） 查找某个字符是否存在于某个数组中  存在返回下标（首次出现的位置）  否则返回-1  
lastIndexOf （） 查找某个字符在某个字符串中 最后一次出现的位置  


substr(start,length)   返回从开始位置截取指定长度的字符串  如果只有一个start，默认截取到最后
substring(start,end)   返回从开始位置截取到结束位置处的字符  不包括结束位置处的字符

实例：  已知一个排好序的字符串，向这个字符串中添加一个字符，添加后保证顺序不变  
adefh   插入  c
acdefh
思路： 得到 c所在位置处的前面字符串 和  后面字符串，在进行字符串的拼接  


toLowerCase()   转小写   toUpperCase()  转大写
trim()  去掉字符串两端空白字符  

replace(原字符串，新字符串)    返回的结果是替换后的新字符串   默认只能替换一个   全部替换（正则）
