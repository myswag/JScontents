正则表达式相关方法

test() 检验一个字符串是否符合某一个正则规范，如果符合返回true，否则返回false
用法： reg.test(str)
----------------------------------------
exec()  根据正则表达式查找，结果会返回一个长度为1的数组 （数组只有一个值）
用法： reg.exec(str);
 


正则中字符串的方法：
-----------------------------------------
search方法     返回与正则表达式查找内容匹配的第一个子字符串的位置。
 用法：  str.search(reg);
 
---------------------------------------------------------------

match() 方法   使用正则表达式模式对字符串执行查找，并将包含查找的结果作为数组返回。
 用法：  str.match(reg) 
         
 说明：正则如果有g  表示全局匹配，返回一个数组
          正则如果没有g   表示match只能在str中匹配一次，返回的第一个元素是匹配的正则结果，其余的元素是  正则中子表达式匹配的结果
    
-------------------------------------------------------

str.replace(原字符串,新字符串)

replace（）   通过正则表达式替换，返回结果 是原字符串被替换后的新字符串  

用法：  str.replace(正则表达式,要替换的新字符串)      如果想实现全局替换，在正则表达式上加一个修饰符 g 


正则表达式方法练习

    str = "good good study,day day up";  //将该字符串中的 good  替换成 best
    reg = /good/g;
    str.replace(reg, "best" );   

    str = "  hello  big  , hello  big beg bog bag bag   ";
    // 将bog  beg  bag  big  替换成 bug
    reg = /b[oeai]g/g; 
    str.replace( reg,"bug" )


    //去除字符串开始位置处的空白字符
    "   he  llo  ".replace(/^\s+/,"");

    //去除字符串结束位置出的空白字符
      "   he  llo  ".replace(/\s+$/,"");

    //一次性去除字符串开始和结束位置处的空白
       "   he  llo  ".replace(/(^\s+)|(\s+$)/g,"");

    //将字符串中的所有空白字符去除
    "   he  llo  ".replace(/\s+/g,"");

严格模式 ：  "use strict"  
严格模式下几个要求：
      变量必须要用var 定义  
      严格模式下 八进制 被禁用
           严格模式下 arguments和形参不同 （非严格模式下  形参值的改变会影响arguments，严格模式下 形参的值改变不会影响实参副本）
      函数必须声明在最顶层

绑定this指向 

1 、bind()   绑定匿名函数中的this指向 
     setTimeout(function(){
          //alert( this );
          this.style.fontSize = "100px";

     }.bind(document.getElementById("d")),1000)

2、 apply() 和 call()   绑定函数中的this指向
 function fun(){
     alert( this );
 }
 fun.call( document );
 fun.apply( document );

let 块级作用域 

1、let不能重复定义变量
2、let必须作用在某个{}内  let只在某个{} 内起作用
     if(true){
          let num = 90;
          alert(num);
     }
     alert( num );
3、使用let定义的变量，该变量没有定义前不能被使用（暂时性死区）
var num = 90;
let num = 80;//报错
4、let不能实现变量提升
