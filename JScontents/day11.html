1、事件对象  event

1、什么是事件

 对网页元素的操作，就称为事件   onclick/ onmouseover/.......

2、什么是事件对象
 当对页面元素发生某个事件操作时，会产生一个事件对象。
 事件对象兼容 ：  evt || window.event;
 不同的事件对象 对应属性 不一样 

2、鼠标坐标属性

pageX  /  pageY    距离文档窗口的距离
clientX / clientY  距离浏览器窗口的距离  当页面没有滚动条时 结果和pageY一样
offsetX / offsetY  距离目标元素的内部偏移
screenX / screenY  距离屏幕窗口的距离

3、button属性

值有三个：
左键： 0 

滚轮： 1

右键： 2

IE6的结果是 1   2   4
封装一个函数，兼容 ： button属性：
function getButton(evt){
     var e = evt || event;
     if( evt ){ //高版本
          return e.button;
     }else if( window.event ){ //低版本
          switch( e.button ){
               case 1:return 0;
               case 4:return 1;
               case 2:return 2;
          }
     }
}

5、事件流

在一个事件触发时，程序从 内到外  或 从外到内部执行的一个过程，称为一个事件流

事件流两种模式：
     事件冒泡：事件从子元素向父元素触发  （在某个事件被触发时，相同的事件会向父元素触发，这个过程就叫事件冒泡）  
              冒泡过程 ： html元素 --- body ---document --- window（ie6没有window对象）
     事件捕获：事件从父元素向子元素触发

10、事件监听  addEventListener()

好处 ： 
     可以为同一个对象绑定多个同样的事件  
     根据监听确定是执行冒泡还是执行捕获
用法：
     obj.addEventListener( "去掉on的事件名称",事件处理程序（匿名函数）, 布尔值 )   默认是冒泡 false
