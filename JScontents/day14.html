1、cookie生存期

cookie信息在浏览器上保存的时间
如果没有设置生存期，浏览器关闭，cookie信息自动消失 
如果设置了生存期 ，cookie信息会在设置时间之内保存在浏览器上

如何设置生存期： 
document.cookie = "键=值;expires=几天后的时间"   几天后的时间 是标准时间格式 

删除生存期 ： 
将值改为 ""  或  生存期设置为 -1

2、cookie注意的问题

1、cookie不能跨域访问  在一个文件下不能看到另一个文件夹下的cookie信息
2、cookie存储少量的数据，大小不能超过 4kb
3、cookie 存储个数 不能超过50个
4、cookie参数 ：
document.cookie = ""
键=值           必选
expires 生存期  可选
path  路径   可选
domain   域名  可选
secure  安全   值有 0 和 1 两种情况  默认 1  （表示 在https或http下都可以访问 ）

元字符：
^  表示正则开始
$  表示正则结束
.  表示任意一个字符 
*  等价  {0,} 表示前面紧挨着的字符至少0个  至多无数
+  等价  {1,}  表示前面紧挨着的字符至少1个
?  等价  {0,1}  
\  转义字符   当有一些特殊字符在正则中表示自身含义是，需要转义  \+  表示 +     \. --- .
() 优先级最高  
|   表示或者   


组合字符 ： 
   \w   : 表示一个单词字符（字母或数字或下划线）
   \W   : 表示非单词字符 （除了字母  数字 下划线之外的字符）
   \d  : 等价 [0-9]  任意一个数字
   \D  : 等价 [^0-9] 非数字
   \s : 表示一个空白字符 （tab、space）
   \S : 表示非空白字符


{} 表示范围

{m} m代表正整数 表示括号前面紧挨着的字符只能出现m个  
{m,n} m < n  m、n都是正整数    表示括号前面紧挨着的字符至少出现m个，至多出现n个
 {m, }   表示括号前面紧挨着的字符至少出现m个 没有上限

[  ]  : 表示括号内的任意一个字符   
[2&rew]  [a-z_3]
[^  ]  : 表示除了括号内的任意一个字符
[^a-z] 除了小写字母
[^0-9]
